# DriveLeads - Lead Management System for HSR Motors

A modern, intuitive lead management web application designed for HSR Motors' sales team and business managers. DriveLeads replaces manual spreadsheet tracking with a collaborative, real-time platform featuring smart automation, lead scoring, and analytics.

## Product Overview

**Product Name:** DriveLeads
**Tagline:** "Drive Every Lead to a Sale"
**Target Users:** HSR Motors Sales Team & Business Managers

## Screens

### Screen 1 - Dashboard (Business Manager View)
- KPI cards: Total Leads, Conversion Rate, Avg Response Time, Pipeline Value
- Lead trend charts (weekly/monthly/yearly)
- Lead source breakdown (donut chart)
- Conversion funnel visualization
- Top sales reps leaderboard
- Real-time activity feed

### Screen 2 - Lead Listing (Sales Team View)
- Sortable, filterable table of all leads
- Filters: Status, Source, Assigned Rep, Date Range
- Quick actions: Call, Email, View Details
- Bulk actions: Assign, Change Status, Export
- Lead score indicators (Hot/Warm/Cold)
- Pagination & search

### Screen 3 - Lead Pipeline (Kanban Management)
- Visual Kanban board: New → Contacted → Qualified → Negotiation → Won
- Drag-and-drop status updates
- Smart alerts (auto-assign warnings, follow-up reminders, hot lead alerts)
- Budget and vehicle interest on each card
- Pipeline statistics

### Screen 4 - Lead Details
- Complete contact information
- Vehicle interest details (model, budget, timeline, preferences)
- Activity timeline with full audit trail
- Lead score breakdown with factor analysis
- Quick action buttons (Call, Email, SMS, Schedule)
- Assigned rep info & reassignment
- Upcoming action items
- Notes section
- Quick status update buttons

## Key Automation Features

1. **Auto Lead Assignment** - Round-robin distribution to sales reps based on availability and workload
2. **Smart Lead Scoring** - Automatic 0-100 scoring based on budget match, purchase timeline, engagement, and location
3. **Automated Welcome Messages** - Instant SMS/email acknowledgment when a lead is captured
4. **Follow-up Reminders** - SLA-based alerts ensuring no lead goes uncontacted
5. **Hot Lead Alerts** - Real-time notifications for high-score leads requiring immediate attention
6. **Duplicate Detection** - Prevents duplicate entries from multiple ad sources
7. **Auto Status Updates** - Status suggestions based on activity patterns

## Tech Stack (Prototype)

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js for data visualizations
- Font Awesome icons
- Inter font family
- Fully responsive desktop design

## How to View

Open `index.html` in any modern web browser. No build tools or server required.

## Design Decisions

- **Color System**: Blue primary (#4F6EF7) conveys trust and professionalism; green for success states; orange for warnings
- **Typography**: Inter font for clean, modern readability
- **Layout**: Fixed sidebar navigation for quick switching between views
- **Data Density**: Optimized for business users who need to scan large amounts of data quickly
- **Automation Visibility**: Automated actions are clearly labeled so users understand what the system does vs. manual actions
